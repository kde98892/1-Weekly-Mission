- [x]  “공유” 버튼을 누르면 “폴더 공유” 모달이 보이나요?
- [x] “이름 변경” 버튼을 누르면 “폴더 이름 변경” 모달이 보이나요?
- [x]  “삭제” 버튼을 누르면 “폴더 삭제” 모달이 보이나요?
